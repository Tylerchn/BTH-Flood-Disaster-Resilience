# 京津冀城市群洪涝韧性全过程可视化仪表盘

Beijing-Tianjin-Hebei Urban Agglomeration Flood Disaster Resilience Dashboard

## Overview

基于DPSIR框架的京津冀城市群洪涝灾害韧性"评估—诊断—优化"全过程交互式可视化平台，集成以下三个实证章节：

- **第四章** 灾前韧性基线的双维评估与融合诊断（节点功能韧性 + 网络联系韧性）
- **第五章** 洪涝风险的时空格局与耦合驱动机制（HESA四维风险 + 风险韧性四象限）
- **第六章** 灾后协同恢复优化与差异化策略（Dueling DDQN + 三队伍修复动画）

## Tech Stack

- React 18 + Vite
- 纯 SVG 可视化（无外部地图库依赖）
- GitHub Pages 自动部署

## Quick Start

```bash
npm install
npm run dev
```
